# Sintética — Weekly Strategic Rhythm

<!-- Organizational discipline tracking. The heartbeat checks these weekly.
     Inspired by habit-tracking logic: one intentional action per pillar per week. -->

## Weekly Pillars

1. **Signal Watch** — Log at least one new entry in `sources/trends/` or `sources/future-signals/`
   - Auto-detectable: new file created in either domain this week via git log

2. **Hypothesis Update** — Review or create one entry in `sources/hypotheses/`
   - Auto-detectable: new or modified file in `sources/hypotheses/` this week

3. **Decision Log** — Capture at least one key decision in `sources/decisions/`
   - Requires self-reporting: decisions are not detectable from file activity alone

4. **Opportunity Scan** — Review or add one entry in `sources/opportunities/`
   - Auto-detectable: new or modified file in `sources/opportunities/` this week

5. **Lesson Capture** — Distill one lesson from active project work into `sources/lessons/`
   - Requires self-reporting

## Heartbeat Behavior

- **Monday 9 AM**: Check which pillars triggered since last Monday.
  For untriggered pillars, suggest one specific action based on active projects.
- **Friday 5 PM**: Summarize the week's rhythm.
  If all 5 pillars triggered: generate a "Weekly Strategic Pulse" in `drafts/active/`.
